# W2D2 Assignment: Temperature Converter

### Your Mission
You must write two custom methods **outside** of your `main` method block.
1. `celsiusToFahrenheit(double c)`: Converts Celsius to Fahrenheit and returns the result.
2. `fahrenheitToCelsius(double f)`: Converts Fahrenheit to Celsius and returns the result.

**Beware the integer math trap:** `5/9` in Java is `0`! Use `5.0/9.0` to force double division.
